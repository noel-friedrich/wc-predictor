# Read a QR by Hand

> A simple web-tool to predict the knockout stage of the 2026 FIFA World Cup  
> Companion YouTube video: [youtu.be/BjYtM6TFlbc](https://youtu.be/BjYtM6TFlbc)  
> Hosted at [noel-friedrich.de/wc-prediction](https://noel-friedrich.de/wc-prediction)

## About

I made a simple mathematical model to predict the knockout phase of the 2026 FIFA World Cup. It might not be very accurate, but it's good at giving an intuition for counter-intuitive tournament probabilities.

## How to use

- Open the hosted version [here](http://noel-friedrich.de/wc-prediction)
- or host it yourself by statically hosting the `index.html` file using your favourite host.

## Status

The website will be updated to keep track of the progress in the knockout stage of the 2026 FIFA World Cup.
